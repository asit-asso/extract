Le plugin d'extraction QGIS Server permet d'imprimer un Atlas QGIS Server au format PDF

Important :

  • Si le paramètre 'CRS (Code EPSG)' n'est pas renseigné en config alors la valeur par défaut EPSG:2056 sera utilisé
  • Si le paramètre 'Layers à utiliser' n'est pas renseigné alors, le paramètre LAYERS ne sera pas transmis dans l'appel du WMS de QGIS Server
  • L'impression PDF pourrait échoué si une quantité trop importante d'entités est demandé, le projet QGIS utilisé définit le nombre max d'entités à imprimer
